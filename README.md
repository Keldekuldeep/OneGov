# OneGov Portal

A modern, SEO-friendly government portal built with Next.js, TypeScript, Tailwind CSS, and Shadcn UI.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Shadcn UI** - High-quality component library
- **Lucide React** - Beautiful icons

## Why This Stack?

✅ **SEO Friendly** - Server-side rendering for government pages
✅ **Fast Development** - Pre-built components with Shadcn UI
✅ **Type Safe** - TypeScript prevents bugs
✅ **Dashboard Ready** - Perfect for admin panels
✅ **Spring Boot Compatible** - Easy API integration
✅ **Quick Deploy** - Vercel deployment in 2 minutes

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```

## Deploy to Vercel

```bash
vercel
```
